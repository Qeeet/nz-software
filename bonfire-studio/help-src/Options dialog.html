The Options gialog is the central configuration engine for the whole application. It contains the following 3 configuration tabs:
  • General - Configure general application options.
  • Project Bar - Configure options associated with project management.
  • Editor - Configure the fonts and colors of the text editor.
  • XML Tree - Configure the XML Tree.
  • Views/Parsers - Configure the views and the syntax-highlighting options for each file type.
Click on the above links to find out more information about the specific configuration options.
