List of icons in the Edit Menu
  • Undo - Undo the last typing action.
  • Redo - Undo the next typing action.


  • Cut - Cut selected text and copy to clipboard.
  • Copy - Copy selected text and copy to clipboard.
  • Paste - Paste clipboard text into current location in the text file.
  • Select All - Select all text of the current file.
  • Delete - Delete the current character.


  • Tab Selection - Indent the selection by one tab.
  • Untab Selection - Unindent the selection by one tab.
  • Uppercase Selection - Convert all characters in the selection to uppercase.
  • Lowercase Selection - Convert all characters in the selection to lowercase.


  • Read Only - Displays the Read-Only flag of the current file.
