List of icons in the View Menu
  • Toolbars
    • Main Toolbar - Show/Hide the Main Toolbar.
    • Project Toolbar - Show/Hide the Project Toolbar.
    • Find Toolbar - Show/Hide the Find Toolbar.

  • Project Manager - Show/Hide the Project Manager window.
  • Output - Show/Hide the Output window.
  • Status Bar - Show/Hide the Status Bar.


  • Fullscreen - Removes all toolbars to give the user a "Full Screen" to work with.
