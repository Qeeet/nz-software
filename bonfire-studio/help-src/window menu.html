List of icons in the Window Menu
  • Cascade - Shows the document windows arranged like a deck of cards.
  • Tile Horizontally - Tiles all document windows horizontally.
  • Tile Vertically - Tiles all document windows vertically.
  • Arrange Icons - Search the next occurence of the search text.


  • Close Current Window - Close the current document window.
  • Close All Windows - Close all document windows.


  • Current Windows - Shows the list of the current document windows.
