The MiniCalc menu items are described below:
  • Calc
    • Always on Top - Keep the MiniCalc window above all other windows (except other Always-on-Top windows).
    • Translucent - On Windows 2000 and later, this command will make the main window translucent, so other windows that are below MiniCalc could be viewed.
    • Auto-Copy Result to Clipboard - When a calculation is successful, the result will automatically be copied to the Windows Clipboard (erasing all previously stored Clipboard content).
    • View Symbol Table - Display a dialog box containing a listing of all default and user-created functions and variables.
    • Options - Show a dialog box containing options about trigonometric functions and other user preferences.
    • Exit - Exit MiniCalc.
  • Help
    • Contents - Displays the help contents for MiniCalc.
    • About MiniCalc - Displays information about MiniCalc including the copyright information and product homepage.
