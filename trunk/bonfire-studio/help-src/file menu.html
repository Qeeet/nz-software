List of icons in the File Menu
  • New File - Create a new text file.
  • Open File - Open an existing file from disk.
  • Open from URL - Downloads and opens a file from a specified URL.


  • Save File - Save the current file to disk.
  • Save File As - Save the current file with a different filename.
  • Save All - Save all modified files and projects.


  • Print Preview - Preview the current docuemnt before printing.
  • Print Setup - Setup printers.
  • Print - Pops up the print dialog.


  • File Properties - Displays a popup with the information about the current file.


  • Recent Files (submenu) - Shows a list of recently opened files.
  • Recent Projects (submenu) - Shows a list of recently opened projects.


  • Exit - Close the application.
