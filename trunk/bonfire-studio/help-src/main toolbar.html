List of icons in the Main Toolbar
  • New File - Create a new text file.
  • Open File - Open an existing file from disk.
  • Save File - Save the current file to disk.
  • Save All - Save all modified files and projects.


  • Print - Pops up the print dialog.


  • Undo - Undo the last typing action.
  • Redo - Undo the next typing action.


  • Cut - Cut selected text and copy to clipboard.
  • Copy - Copy selected text and copy to clipboard.
  • Paste - Paste clipboard text into current location in the text file.


  • Fullscreen - Removes all toolbars to give the user a "Full Screen" to work with.


  • Help - Pops up the Help/About dialog.
