List of actions in the Project Menu
  • Refresh Project - Refreshes the project tree.


  • New Project - Create a new text file.
  • Open Project - Open an existing file from disk.


  • Save Project - Save the current file to disk.
  • Save Project As - Save the current file with a different file name.


  • Add Files - Add file(s) to project.
  • Add Folders - Add folder(s) to project.
  • Remove Item - Remove current file or folder from the project.


  • Project Properties - Displays a popup with the properties of the project file.


  • Close Project - Close the current project and save it if needed.
