List of icons in the Search Menu
  • Find - Pops up a search dialog for the user to input search criteria to find text in the current file.
  • Find Previous - Search the previous occurence of the search text.
  • Find Next - Search the next occurence of the search text.
  • Replace - Replace the search text with the replace text.
  • Find in Files - Find the search text inside files in specified folders.


  • Goto Line - Go to a specific line in the current file.
