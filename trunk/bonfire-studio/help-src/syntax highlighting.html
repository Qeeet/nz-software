Syntax highlighting is a tool that shows different sections of a text file in different colors. For example in XML files, you might see the tag name in BROWN, while seeing plain text in BLACK. Here is an example of a C++ file using syntax highlighting:
//--------------------------------------------------------------------
BOOL ModuleTerminate(CThisTestModule * pThisTestModule)
{
	// Drop the table created in the ModuleInit
	if( pThisTestModule->m_pVoid )
	{
		// Remove table from database and Delete CTable object
		((CTable *)pThisTestModule->m_pVoid)->DropTable();
		delete ((CTable *)pThisTestModule->m_pVoid);
		pThisTestModule->m_pVoid = NULL;
	}
	
	return ModuleReleaseDBSession(pThisTestModule);
}
  • To configure the colors for syntax-highlighting click on Options/Editor
  • To configure the type of syntax-highlighting parser to use for each type of file click on Options/Views
