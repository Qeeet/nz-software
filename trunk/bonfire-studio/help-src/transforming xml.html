To transform an XML document with XSL, follow these simple steps:
  • Create the XML file.
  • Create the XSL file.
  • Specify the XSL "link" within your XML file like this:
    <?xml-stylesheet type="text/xsl" href="myxslfile.xsl" ?> right after the <?xml version="1.0"?> declaration.
  • Go to Browser view in Bonfire Studio and it will show you the outcome.
