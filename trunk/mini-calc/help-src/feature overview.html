Version 1.0
  • Always On Top (keeps MiniCalc on top of all other programs)
  • Translucent Mode (keeps MiniCalc half visible, allowing viewing of windows under the MiniCalc window)*


  • Advanced Mathematical Expression input syntax (e.g. "2 * sin(pi / 2) ^ (2 - x)")
  • Expandable symbol table (unlimited creation of expression-based functions and variables)
  • Basic set of default internal functions (e.g. sin, cos, tan, fact, abs, sqrt, etc.)
  • Customizable degree/radian modes for trigonometric functions
  • Keep created functions and variables when exiting (Save Symbols)
  • Automatically copy the result of an expression to the Windows Clipboard (WARNING: this erases all previously stored clipboard data)


  • Output window in the style of standard graphing calculators
* Note: translucent mode is only available on Windows 2000 or later
